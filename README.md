# Cubism

Cubism ist als rein client-seitige HTML-Webanwendung geplant. Die Anwendung soll als PWA installierbar sein, offline laufen und primär einen `3x3x3`-Rubik's-Cube lösen können. `4x4x4` ist als Erweiterung mit eigener Reduktionslogik vorgesehen.

Der Fokus liegt auf drei Bereichen:

- Erfassung des Würfelzustands durch manuelle Eingabe und später per Kamera/Fotos
- Berechnung eines Lösungswegs vollständig lokal im Browser
- Visuelle Darstellung des Lösungswegs Schritt für Schritt

Die fachliche Basis ist in den Dokumenten unter [`docs/`](/Users/mat/code/Cubism/docs) beschrieben.
